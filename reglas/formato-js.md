4. ⚙ Buenas prácticas
Evitar variables globales
Usar const y let en lugar de var
Mantener funciones cortas y claras
Separar responsabilidades (cada función hace una cosa)
Evitar código duplicado y complejo

5. 🌐 DOM y eventos
Selección de elementos: document.querySelector / document.getElementById
Manejo de eventos: addEventListener
No modificar directamente HTML si se puede actualizar mediante clases o texto
